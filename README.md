# Gestor SQLite con Interfaz Gráfica en Tkinter (Python)

Proyecto de una aplicación de escritorio que permite gestionar una base de datos SQLite mediante una interfaz gráfica. Facilita la administración de **alumnos, cursos, tareas, entregas y asignaciones** sin necesidad de escribir consultas SQL.

---
## Importante
Antes de ejecutar el proyecto, necesitas:
- **Python 3.x**
- ** El proyecto incluye una base de datos de ejemplo llamada `proyecto.db` dentro de la carpeta `/database`. Esta viene con algunos ejemplos de alumnos, cursos, entregas y asignaciones. No es necesario configurar nada adicional en cuanto a la base de datos fuera del programa.**
---
# Cómo usar
 
### 1️. Descargar y descomprimir el proyecto  
**Descarga el archivo ZIP** y **descomprímelo**. Te quedará una carpeta con todos los archivos necesarios.
<img src="https://i.imgur.com/PIaYLiP.png" width="550"> <img src="https://i.imgur.com/FImIcvh.png" width="200">


---

### 2️. Entra en la carpeta del proyecto  
Dentro de lo que has descomprimido, abre la carpeta:

**`SQLite_Management-master`**

---

### 3️. Busca el archivo que inicia el programa  
Dentro de esa carpeta verás:

**`main.py`**

Ese es el archivo que abre la aplicación.

---

##  Opciones para abrir el programa

###  Opción A — Doble clic en `main.py`
Si tienes Python instalado correctamente:  
**Haz doble clic en `main.py` y el programa se abrirá.**

Si no se abre, usa la opción B.

---

###  Opción B — Ejecutarlo desde la consola
1. Abre la consola en la dirección de la carpeta:
   - **Windows:** busca “CMD” y escribe después: `cd [Ruta-de-la-carpeta]`
   - **Mac/Linux:** abre Terminal y escribe después: `cd [Ruta-de-la-carpeta]`
* Si no sabes como buscar la ruta de la carpeta siempre puedes abrir la consola en la carpeta directamente haciendo clic en un espacio vacío y en *'Abrir en Terminal'* o *'Nueva terminal en la carpeta'*:
<img src="https://i.imgur.com/blxCWuK.png" width="550">
2. Escribe este comando:

```bash
python main.py
```
