# Gestor SQLite con Interfaz Gráfica en Tkinter (Python)

Proyecto de una aplicación de escritorio que permite gestionar una base de datos SQLite mediante una interfaz gráfica. Facilita la administración de **alumnos, cursos, tareas, entregas y asignaciones** sin necesidad de escribir consultas SQL.

---
## Construido con

* [**Python**](https://www.python.org/) – Lenguaje principal
* [**Tkinter**](https://docs.python.org/es/3/library/tkinter.html) – Interfaz gráfica
* [**SQLite**](https://www.sqlite.org/) – Base de datos
* [**Git**](https://git-scm.com/) – Control de versiones

---
## Pre-requisitos

Antes de ejecutar el proyecto, necesitas:

- **Python 3.x**
- Archivo de base de datos **'proyecto.db'**
- Configurar la ruta **"DB_RUTA"** en **'funciones.py'** para que apunte al **'proyecto.db'**
